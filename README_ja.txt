Assetto Corsa Rally セーブバックアップ
======================================

概要
----
Assetto Corsa Rally のローカルセーブデータ（.sav）を、選択したバックアップ先フォルダへ複製するためのWindows向けGUIアプリです。
元のセーブデータには変更を行いません。

主な機能
--------
- セーブデータ格納フォルダを選択できます。
  例: C:\Users\User\App Data\Local\acr\Saved\SaveGames
- バックアップ先フォルダを選択できます。
- バックアップ実行時に、日時入りの新規フォルダを作成します。
  例: ACR_SaveBackup_20260707_213045
- セーブデータ格納フォルダ直下の .sav ファイルを、新規バックアップフォルダへ複製します。
- インターフェイス言語は日本語と英語に対応しています。
- 一度選択した各フォルダはユーザーごとに保存され、再選択するまで次回起動時にも引き継がれます。

使い方
------
1. Run_ACRSaveBackup.bat をダブルクリックします。
2. 「セーブデータ格納フォルダ」を選択します。
3. 「バックアップ先フォルダ」を選択します。
4. 「バックアップ実行」を押します。

フォルダ設定の保存
------------------
選択したセーブデータ格納フォルダとバックアップ先フォルダは、以下に保存されます。

%APPDATA%\ACRSaveBackup\settings.json

ユーザーが別のフォルダを選択するまで、次回起動時にも同じフォルダが使用されます。

実行ポリシーについて
--------------------
Run_ACRSaveBackup.bat は、同梱の PowerShell スクリプトを ExecutionPolicy Bypass で起動します。
これはこのアプリの起動時のみ適用され、Windows全体の設定を変更するものではありません。

注意事項
--------
- 複製対象は、選択したセーブデータ格納フォルダ直下にある .sav ファイルです。
- バックアップ先フォルダには、セーブデータ格納フォルダとは別の場所を指定してください。
- バックアップ前にゲームを終了しておくことを推奨します。
